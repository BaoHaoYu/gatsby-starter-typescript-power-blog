| 功能               | 快捷键                 |
| ------------------ | ---------------------- |
| 格式化整个文件     | SHIFT + ALT + F        |
| 格式化选中文本     | CTRL + K + F           |
| 折叠所有           | CTRL + K + 0           |
| 展开所有           | CTRL + K + J           |
| 折叠第一级         | CTRL + K + 1           |
| 折叠第二级         | CTRL + K + 2           |
| 重命名             | F2                     |
| 找到所有引用       | SHFIT + F12            |
| 切换标签           | CTRL + TAB             |
| 显示隐藏底部栏     | CTRL + J               |
| 跳到某一行         | CTRL + G               |
| 根据文件名打开文件 | CTRL + P               |
| 命令               | F1 or SHIFT + CTRL + P |
| 首选项             | CTRL + ,               |
